# Lista della Spesa PWA

Questo progetto è una Progressive Web App (PWA) per registrare e gestire la lista della spesa.

## Funzionalità iniziali
- Login utente (pagina /login)
- Interfaccia semplice con CSS standard

## Avvio

1. Installa le dipendenze:
	```bash
	npm install
	```
2. Avvia il server di sviluppo:
	```bash
	npm run dev
	```

Apri [http://localhost:3000/login](http://localhost:3000/login) per vedere la pagina di login.

- Aggiungere la gestione della lista della spesa
